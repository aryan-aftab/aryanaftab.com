Raspberry Pi Reverse Proxy

Initially, I faced the challenge of accessing internal services outside my local network. To address this, I embarked on a project to create a small reverse proxy using a Raspberry Pi 5.

My ISP unfortunately was not able to provide me with a static IP, which is required in the first place to host services, since I was behind a CGNAT. However, one solution to this was to connect the Raspberry Pi to a cloud VPS provider via WireGuard, which a highly secure state-of-the-art VPN protocol.

Utilizing this setup in conjunction with iptables enabled me to assign a static IP to the Raspberry Pi, facilitating remote access from external networks and allowing services to be seamlessly forwarded to its ports.

This setup not only enhances security by isolating internal networks but also refines my network administration and security skills, demonstrating the power of innovative network solutions.

Ergonomic Keyboard

Embarking on the development of a custom split keyboard was a highly rewarding endeavor that seamlessly integrated my passion for ergonomics and efficient typing. The design drew inspiration from the Halmak layout, renowned for its optimized key placement that minimizes finger travel and enhances typing speed.

To achieve a truly ergonomic configuration, I utilized KiCad to modify open-source PCB designs, enabling the creation of a staggered layout that aligns perfectly with my hand's natural resting position, ensuring both comfort and accessibility.

Subsequently, I compiled and flashed the firmware using QMK, an open-source keyboard firmware renowned for its extensive customization capabilities. After extensive experimentation with the layout, I achieved a remarkable ~30% increase in typing speed, elevating my initial 120 WPM to 150 WPM.
